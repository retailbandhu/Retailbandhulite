## üÜò **TROUBLESHOOTING**

### **Calculator not opening:**
- Try clicking icon again
- Refresh the page
- Check if another modal is open

### **CSV import failing:**
- Check CSV format (use template)
- Verify required fields (Name, Price, Stock)
- Look at error messages for specific issues
- Make sure prices are numbers, not text

### **Dark mode not saving:**
- Check browser localStorage enabled
- Try toggling again
- Clear browser cache if needed

### **Date picker not applying:**
- Make sure you clicked "Apply" button
- Check if both dates are selected
- Try using a preset first

### **Export not downloading:**
- Check browser download settings
- Allow popups/downloads for this site
- Try a different browser
